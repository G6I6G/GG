# Морской бой (Node.js + Socket.io)

Простой реал-тайм Морской бой: фронтенд можно разместить на GitHub Pages, бэкенд — на Heroku/Vercel или любом другом Node-сервере.

## Локальный запуск

```bash
npm install
npm start
```

Сервер поднимется на `http://localhost:3000` и будет отдавать файлы из `public/`.

## Деплой

- **Frontend (GitHub Pages):** публикуйте содержимое `public/` на Pages. В поле «URL сервера» в UI укажите домен, где запущен Node.js сервер.
- **Backend (Heroku/Vercel):** задеплойте `server.js` с зависимостями `express` и `socket.io`. Для Socket.io откройте веб‑сокеты и разрешите CORS (в коде уже установлено origin: "*").

После деплоя клиент соединится с сервером по указанному URL и автоматически поставит игрока в очередь на матч.
