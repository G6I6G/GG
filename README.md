# Discord Voice Concierge

Веб-приложение на Flask с интегрированным Discord-ботом, который создаёт приватные голосовые каналы, отправляет персональные приглашения и удаляет каналы, когда приглашение истекло или все ушли.

## Быстрый старт
1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
2. Задайте переменную окружения с токеном бота:
   ```bash
   export DISCORD_BOT_TOKEN="<ваш токен>"
   ```
3. Запустите приложение:
   ```bash
   python app.py
   ```
4. Откройте страницу http://localhost:5000 и заполните форму. Бот отправит приглашение выбранному пользователю в личные сообщения.

## Ключевая логика
- Канал создаётся видимым для всех, но с запретом на подключение по умолчанию.
- Принимающий пользователь нажимает кнопку в DM, получает право подключения и одноразовую ссылку на канал.
- Если канал пуст после ухода приглашённых или срок приглашения истёк, бот удаляет канал и очищает данные.

## Переменные окружения
- `DISCORD_BOT_TOKEN` — токен вашего бота (обязательно).
- `PORT` — порт для Flask (необязательно, по умолчанию `5000`).
